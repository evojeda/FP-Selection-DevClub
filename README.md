## FP Selection ##

O FP Selection é uma aplicação web desenvolvida como parte das atividades do DevClub, com foco em praticar conceitos de desenvolvimento front-end, organização de código e boas práticas de estruturação.

A proposta do projeto é apresentar uma interface moderna e funcional, aplicando conceitos de estilização, interatividade e responsividade.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript 

## Objetivo do projeto

O principal objetivo foi colocar em prática:

- Estruturação de páginas web
- Organização de estilos
- Manipulação do DOM com JavaScript
- Separação de responsabilidades (HTML, CSS e JS)
- Publicação de projeto em produção (GitHub Pages)

Além disso, o projeto também serviu como treino para manter um código mais limpo, organizado e escalável.

## Interatividade com JavaScript

O JavaScript foi utilizado para trazer dinamismo à aplicação, permitindo interações com o usuário e manipulação de elementos da página.

Com ele, foi possível:

- Controlar comportamentos da interface
- Responder a ações do usuário
- Tornar a experiência mais fluida e interativa

Ou seja, ele é o cérebro por trás das ações da aplicação

## Diferencial na estrutura do CSS

Um dos pontos mais interessantes desse projeto foi a forma como o CSS foi estruturado.

Ao invés de declarar todas as classes separadamente em linhas independentes, foi utilizada uma abordagem mais organizada e encadeada, utilizando:

& + .classe

Essa estratégia permite criar uma relação mais clara entre os elementos e seus estilos, deixando o código:

- Mais legível 
- Mais organizado 
- Mais fácil de manter 

Essa abordagem é muito comum em metodologias mais modernas de escrita de CSS, pois ajuda a evitar repetição e melhora a hierarquia visual do código.
